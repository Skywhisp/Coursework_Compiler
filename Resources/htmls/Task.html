Постановка задачи 

Целочисленная константа с инициализацией на языке Kotlin — это значение, которое представляет собой целое число и назначается переменной при её создании. 
Она позволяет задать начальное значение переменной сразу после её объявления.
Формат записи: 
    const val ИмяПеременной: Int = Число; 
где:
    *const - обязательное ключевое слово, обозначающее, что это константа.
    *val - обязательное ключевое слово, указывающее, что это значение нельзя изменить после инициализации.
    * ИмяПеременной - имя переменной(константы), которое должно быть уникальным в пределах области видимости.
    *Int – тип данных, в данном случае целочисленный.
    *Число – значение, которое присваивается константе при инициализации.
        
Примеры:
    1.	const val a: int = 10; 
    2.	const val b: int = 250;
